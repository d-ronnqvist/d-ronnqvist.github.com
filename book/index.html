3D Graphics with SceneKit

This is the website for the upcoming book, 3D Graphics with SceneKit. It is a book about Apple’s 3D rendering framework for Mac OS X.

Current table of contents:

  • Starting out – From setting up a project to getting a 3D scene on screen.
  • Why things look the way they do – Covers how lights and materials cause geometry to look shaded.
  • More of a Scene – How to use the node hierarchy to create a scene with more geometry, text and animations.
  • Adding detail with images – Using textures, how they work and the effects they can bring.
  • From a point of view – Cameras, looking around in a scene, zooming in on objects.
  • Special effects – A chapter dedicated to effects.
  • Generating geometry – How to generate geometry data to use in SceneKit.
  • Morphing – Setup and morph geometries between multiple targets.
  • Skinning – Bend and move 3D models rigged to skeletons.
  • Performance – How to find and solve common performance problems.
  • The rendering pipeline – An overview of the underlying rendering pipeline.
  • GLSL – An introduction to the OpenGL Shading Language.
  • Custom shading – How to shade geometry using custom shaders.
  • Shader modifiers – Inject your shader code at certain points to make modifications.
  • OpenGL and the render delegate – Using OpenGL to do completely custom rendering.

Chapter names and content is not final and many smaller topics are covered within these chapters but are not mentioned above.

The book is written by me, David Rönnqvist and will only be available as an iBook from the iBooks Store. I’m excited over the possibilities with interactive books and having iBooks on OS X Mavericks means that many readers will be able to experience the book right next to their IDE.

Q&A

I’m a iOS developer, is this books not for me?

SceneKit is not available for iOS but if you want to learn about 3D graphics then I’m not going to stop you.

Being a Mac developer is a not a prerequisite. In fact, I’ve done quite a lot of iOS development myself. As long as you are familiar with Objective-C, Foundation and know a little bit about Core Graphics or Core Animation then you should be fine.

I’ve never done 3D graphics before, will it be to difficult for me?

SceneKit is very easy to get started with and I will explain the concepts behind 3D graphics as the book progresses. You don’t need any 3D experience but it’s good if you have done some basic graphics programming on either OS X or iOS.

I have done 3D graphics before, will this be to basic for me?

Part of SceneKit is the simplicity and part is the integration with other frameworks like AppKit, Core Animation, etc. Even if you’ve done 3D graphics before, using it together with these frameworks will be new and interesting. You may also note that almost a third of the book covers advanced topics like GLSL and OpenGL. I’m sure you’ll find something that you like there.

Will there be maths?

Yes, a little bit. There is always going to be some maths involved and I will do my best to explain it.
