3D Graphics with Scene Kit

This is the website for the upcoming book, 3D Graphics with Scene Kit. It is a book about Apple’s high level 3D rendering framework for Mac OS X and iOS.

The book is written by me, David Rönnqvist and will only be available as an iBook from the iBooks Store. The iBooks format makes it possible for the book to contain rich illustrations, links to sample code, and even interactive 3D models. Having iBooks on both iOS and OS X means that many readers will be able to experience the book right next to their IDE.

Scene Kit was originally introduced at WWDC 2012 and has existed for 2 years on OS X. This year it was made public that Scene Kit is coming to iOS later this fall. Since this isn’t yet released, the first version of the book will cover Scene Kit as it is currently available to the public. Later this year, the book will be updated to also cover Scene Kit for iOS and the new features. Another benefit of being an all digital book is that this new content will become available to you, even if you buy the book before the update.

Apple also introduced a new programming language for iOS and OS X. Since this language is brand new and many Cocoa developers are still used to Objective-C, the examples and sample code in this book will all be written in Objective-C. Next year, the book and the sample code will be updated to also cover programming with Scene Kit in Swift.

Q&A

I’m a iOS developer, is this books not for me (yet)?

While the first version of the book won’t cover iOS specifically, everything you learn is still relevant for both platforms. Later, this book will be updated and you will be able to read about Scene Kit for iOS.

Isn’t Scene Kit just for games?

Some of the new additions to Scene Kit makes it suitable for causal 3D games but that is not all that it’s good for. Part of what I think makes Scene Kit so attractive is how well it integrates with other frameworks. This makes it a good candidate for casual 3D in a 'regular' application as well.

I’ve never done 3D graphics before, will it be to difficult for me?

Scene Kit is very easy to get started with and I will explain the concepts behind 3D graphics as the book progresses. You don’t need any 3D experience but it’s good if you have done some basic graphics programming on either OS X or iOS.

I have done 3D graphics before, will this be to basic for me?

Part of Scene Kit is the simplicity and part is the integration with other frameworks like AppKit, Core Animation, etc. Even if you’ve done 3D graphics before, using it together with these frameworks will be new and interesting. You may also note that almost a third of the book covers advanced topics like GLSL and OpenGL. I’m sure you’ll find something that you like there.

Will there be maths?

Yes, a little bit. There is always going to be some maths involved and I will do my best to explain it. If you have never learnt linear algebra or it was just a long time since you used it, there is an appendix to get you up to speed.
